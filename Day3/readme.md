## 피보나치 수열 만들기

 1. 피보나치 수열을 위해선 기본적으로 두 개의 시작값이 필요함.
 
 2. 두 개의 시작값을 더한 값을 반영하는 변수를 생성하고, 반복문에서 나오는 값이 반영되도록 코드 구성.

 3. 소스코드에서는 for와 while 모두 사용하였으나, 피보나치 수열처럼 증감이 불규칙한 경우 가급적 while을 사용할 것.

 
## 배열

 1. 배열 만드는 방법

  ```java
  //Solution 1
	int arr[] = new int[10]; //10칸 짜리 배열이 생성됨. 값은 들어있지 않음.
  //Solution 2
	int arr2[] = {1,2,3,4,5}; //5칸 짜리 배열이 생성됨. 값은 들어 있음.

  ```

 2. 배열의 단점

  - 배열의 길이가 최초 생성시에 지정되며, 한번 생성되면 변동되지 않는다.

  - 배열을 만들어도 값을 넣기가 귀찮다.
